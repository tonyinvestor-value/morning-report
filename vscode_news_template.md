# 港美股投资晨报生成模板
# 使用方法：选中下面的内容，右键选择"Ask Claude"

请搜索港美股最新投资新闻，严格按要求生成晨报：

## 时效性要求
- 只要最近3天内的新闻（今天、昨天、前天）
- 每条新闻必须标注准确的发布时间
- 如果搜索到4天前或更早的新闻，直接跳过

## 重点关注公司
港股：腾讯(00700)、美团(03690)、理想汽车(02015)、泡泡玛特(09992)
美股：拼多多(PDD)、亚马逊(AMZN)、英伟达(NVDA)

## 股价数据要求
- 如果是交易日盘后：提供当日收盘价 + 涨跌幅 + 涨跌额
- 如果是交易时间内：提供上一收盘价 + 当前价格 + 实时涨跌幅
- 每只股票精确到小数点后2位
- 必须包含：股价、涨跌幅、成交额（如有）

## 信息源限定
重点从以下网站获取：财新网、新浪财经、华尔街日报中文版
